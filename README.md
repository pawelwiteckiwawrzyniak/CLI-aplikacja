# console app

## how to install

```shell
npm install
```

## how to use

### list

![List](https://i.imgur.com/OsT6tpB.png)

```shell
npm run start -- -a list
```

---

### get

![Get by ID](https://i.imgur.com/qYA2WEi.png)

```shell
npm run -- -a get --id
```

---

### add/remove

![Add, remove](https://i.imgur.com/sCm2c3w.png)

```shell
npm run -- -a remove --id
npm run -- -a add --name --email --phone --id
```
